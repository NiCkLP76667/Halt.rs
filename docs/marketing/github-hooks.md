## 📝 Long Description (for GitHub README)

Building multi-agent AI systems? Stop the $500 API bills and debugging nightmares before they happen.

**Halt.rs** is a production-grade Rust proxy that prevents infinite loops, manages task priorities, and provides real-time topology mapping for your AI swarm.

### 🛡️ Key Features

- **Circuit Breaker**: Automatically trips when agents loop infinitely (configurable thresholds)
- **Backpressure Queue**: Priority-based task management (Boss agents always first)
- **Audit Logging**: Real-time swarm topology with token path tracking
- **Multi-Language**: Rust, Python, Java, TypeScript, Go bindings
- **MCP Server**: Model Context Protocol for IDE integration (Cursor, VSCode, Zed)
- **Enterprise**: Docker, CI/CD, REST API, WebSocket, CLI, Plugins

### 🚀 Quick Start

```bash
cargo install halt
halt serve --port 8080
```
